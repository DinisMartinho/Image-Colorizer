# Image-Colorizer

This neural network requires at least **1.1/1.2GB** of **VRAM/RAM** to process a **512x512 image**, with each image taking around **20 to 30 milliseconds** of processing time in Google Collabs **free** Notebooks. It has around **5 million parameters**, trained on **256x256 images**, using a dataset containing **20 thousand** images.
